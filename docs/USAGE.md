# 使用指南

欢迎使用无线旅行 MVP 项目！本文档将帮助你快速上手使用本系统。

## 功能概览

本项目是一个多模块旅行预订平台，包含以下核心功能：

### 1. 用户系统
- 📧 邮箱注册
- 🔐 邮箱登录
- 👤 个人中心
- 📋 订单管理

### 2. 机票预订
- 🔍 搜索国内航班
- 📋 查看航班列表
- 📝 查看航班详情
- ✈️ 填写乘客信息下单
- 💰 模拟支付

### 3. 酒店预订
- 🏙️ 搜索城市酒店
- 🏨 浏览酒店列表
- 📷 查看酒店详情
- 🛏️ 选择房型
- 📅 设置入住天数
- 💳 在线预订

### 4. 火车票预订
- 🚄 搜索火车车次
- 📊 查看不同座位类型
- 💺 选择座位
- 🎫 填写乘客信息购票

## 使用流程

### 首次使用

#### 1. 注册账号
1. 点击右上角"注册"按钮
2. 填写昵称、邮箱和密码
3. 点击"注册"
4. 如果启用了邮箱验证，需要查看邮件并点击验证链接
5. 返回登录页面

#### 2. 登录账号
1. 点击右上角"登录"按钮
2. 输入邮箱和密码
3. 点击"登录"
4. 登录成功后会自动跳转到首页

### 预订机票

#### 步骤 1：搜索航班
1. 在首页选择"机票"标签
2. 选择"单程"或"往返"
3. 填写出发城市（例如：北京）
4. 填写到达城市（例如：上海）
5. 选择出发日期
6. 点击"搜索"按钮

#### 步骤 2：选择航班
1. 在航班列表中浏览可用航班
2. 可以使用筛选功能：
   - 按价格/时间排序
   - 按起飞时间段筛选
3. 点击航班卡片查看详情

#### 步骤 3：填写乘客信息
1. 在航班详情页面填写乘客信息：
   - 姓名
   - 身份证号
2. 可以点击"+ 添加乘客"添加多个乘客
3. 查看价格明细
4. 点击"提交订单"

#### 步骤 4：查看订单
1. 订单创建成功后，自动跳转到"我的订单"
2. 可以点击"立即支付"模拟支付流程
3. 支付后订单状态变为"已支付"

### 预订酒店

#### 步骤 1：搜索酒店
1. 在首页选择"酒店"标签
2. 填写城市名称
3. 选择入住日期和离店日期
4. 可选填写关键词（酒店名称/地标）
5. 点击"搜索"按钮

#### 步骤 2：选择酒店
1. 浏览酒店列表
2. 查看酒店评分、价格、设施
3. 点击"查看详情"进入酒店详情页

#### 步骤 3：选择房型
1. 在酒店详情页面查看：
   - 酒店介绍
   - 设施列表
   - 可用房型
2. 点击房型卡片选择（会高亮显示）
3. 填写预订信息：
   - 入住天数
   - 入住人姓名
   - 联系电话

#### 步骤 4：提交订单
1. 查看价格汇总
2. 点击"立即预订"
3. 跳转到订单页面
4. 可模拟支付

### 预订火车票

#### 步骤 1：搜索车次
1. 在首页选择"火车票"标签
2. 填写出发站（例如：北京）
3. 填写到达站（例如：上海）
4. 选择出发日期
5. 点击"搜索"按钮

#### 步骤 2：选择车次和座位
1. 浏览车次列表
2. 查看不同座位类型和价格
3. 点击座位类型按钮（例如："二等座 ¥553"）
4. 弹出预订窗口

#### 步骤 3：填写乘客信息
1. 在弹窗中填写：
   - 乘客姓名
   - 身份证号
2. 点击"提交订单"

#### 步骤 4：完成预订
1. 订单创建成功
2. 跳转到订单页面
3. 可模拟支付

### 订单管理

#### 查看订单
1. 点击右上角"我的订单"
2. 或在用户头像下拉菜单选择"我的订单"
3. 查看所有订单列表

#### 筛选订单
- **全部**: 显示所有订单
- **待支付**: 仅显示待支付订单
- **已支付**: 仅显示已支付订单
- **已完成**: 仅显示已完成订单
- **已取消**: 仅显示已取消订单

#### 订单操作
- **待支付订单**:
  - 点击"立即支付"完成支付（模拟）
  - 点击"取消订单"取消订单
- **已支付订单**:
  - 等待出行
  - 查看订单详情

### 个人中心

1. 点击用户头像
2. 选择"个人中心"
3. 查看个人信息：
   - 邮箱
   - 昵称
   - 注册时间

## 界面说明

### 首页
- **顶部导航栏**: 快速访问各功能模块
- **搜索区域**: 可切换机票/酒店/火车票搜索
- **热门推荐**: 展示热门城市和特惠信息
- **服务优势**: 平台特色介绍

### 移动端体验
- 所有页面都已适配移动端
- 支持触摸操作
- 响应式布局自动调整

## 数据说明

### 模拟数据
本项目使用模拟数据，所有航班、酒店、火车票信息均为测试数据，不代表真实情况。

### 数据范围
- **航班**: 覆盖北京、上海、广州、深圳等8个主要城市
- **酒店**: 涵盖10个热门城市的多家酒店
- **火车票**: 主要城市间的多个车次

### 价格说明
- 所有价格为示例价格
- 支付流程为模拟，不涉及真实交易

## 技术特性

### 响应式设计
- 基于 TailwindCSS 的响应式布局
- 支持桌面、平板、手机多种设备
- DaisyUI 提供美观的UI组件

### 实时数据
- 使用 Supabase 实时数据库
- React Query 自动缓存和更新数据
- 无需刷新页面即可看到最新数据

### 性能优化
- Vite 快速构建
- 代码分割和懒加载
- 图片和资源优化

## 常见操作技巧

### 快速搜索
1. 在首页使用快捷搜索
2. 点击热门推荐快速跳转
3. 使用筛选功能精确查找

### 多乘客/多房间预订
- 机票支持添加多个乘客

### 订单管理
- 使用状态筛选快速定位订单
- 订单按创建时间倒序排列
- 最新订单显示在最上方
